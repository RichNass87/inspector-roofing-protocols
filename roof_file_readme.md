# Blank Roof Inspection File

This folder was generated from the Inspector Roofing Protocols template.

## Suggested workflow

1. Fill out `inspection.json`.
2. Add roof photos to `photos/`.
3. Update `photos/manifest.csv`.
4. Draft the human-readable report in `reports/roof_report.md`.
5. Review `checklists/report_checklist.md` before delivery.
6. Store PDFs, signed reports, or software exports in `exports/`.

## Folder map

- `inspection.json` - machine-readable inspection file
- `photos/` - source images and photo manifest
- `measurements/` - notes, roof sketches, or measurement exports
- `reports/` - Markdown or generated reports
- `checklists/` - quality-control checklist
- `schemas/` - JSON schema reference
- `taxonomies/` - damage label taxonomy
- `exports/` - generated PDF, CSV, or software exports
