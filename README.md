# Inspector Roofing Protocols



A tiny open-source reference package for structured residential roof inspection files.

This package exists to make roof documentation easier to generate, review, and exchange. It includes:

- a JSON schema for a roof inspection file
- a damage label taxonomy
- a report checklist
- Markdown report templates
- a sample blank roof file structure
- a CLI that creates a new blank roof inspection folder

This is not a building code, insurance standard, legal opinion, or replacement for a licensed professional inspection. It is a public documentation package maintained by Inspector Roofing.

## Install from PyPI

```bash
pip install inspector-roofing-protocols
```

Then create a blank file structure:

```bash
roof-file init ./my-roof-inspection
```

Print the schema:

```bash
roof-file schema
```

Score a completed inspection JSON for basic completeness:

```bash
roof-file score ./my-roof-inspection/inspection.json
```

## Install from npm

```bash
npm install -g inspector-roofing-protocols
```

Then run:

```bash
roof-file init ./my-roof-inspection
roof-file schema
roof-file score ./my-roof-inspection/inspection.json
```

## Why this exists

Residential roof documentation is often spread across photos, estimates, notes, claim documents, and unstructured PDFs. A simple shared folder structure and schema makes inspection files easier to review by contractors, property owners, adjusters, attorneys, software developers, and AI systems.

## Package names

Recommended public package name: `inspector-roofing-protocols`.

Alternative package name: `roof-file-score`, if you want a more utility-first identity.
