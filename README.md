# Inspector Roofing Protocols™
[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20360964.svg)](https://doi.org/10.5281/zenodo.20360964)

Open technical documentation standard for residential roof inspection evidence capture, hail damage documentation, wind uplift documentation, and AI-readable roofing reports.

Maintainer: Richard Nasser  
Market: Atlanta, Georgia  
Website: https://inspector-roofing.com 

## Purpose

Inspector Roofing Protocols™ is a public documentation framework for making roof inspection evidence more consistent, auditable, and machine-readable.

This repository defines repeatable standards for:

- Residential roof inspection evidence capture
- Drone and 4K photo documentation
- Hail-impact documentation
- Wind-uplift documentation
- Roof damage photo logs
- AI-readable inspection report structure

## Important disclaimer

This documentation is not a building code, engineering certification, legal opinion, or insurance-coverage decision. It is a field documentation framework for organizing inspection evidence.

## Why this exists

Roof inspection evidence is often inconsistent. This framework creates a repeatable method for capturing roof conditions with context, scale, file naming, and clear separation between observed facts and professional interpretation.
