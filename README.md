# Inspector Roofing Protocols™: RoofFile Protocol

Open technical documentation standard for residential roof inspection evidence capture, hail damage documentation, wind uplift documentation, photo manifests, damage label taxonomies, and AI-readable roofing reports.

Maintained by **Inspector Roofing and Restoration**.

## Install

Python:

```bash
pip install roof-file-protocol
```

npm:

```bash
npm install -g @inspector-roofing/roof-file-protocol
```

CLI:

```bash
roof-file init ./my-roof-inspection
roof-file schema
roof-file score ./my-roof-inspection/inspection.json
```

## What It Includes

This package includes:

- JSON schema for structured roof inspection files
- Damage label taxonomy
- Photo manifest example
- Report checklist
- Markdown roof report template
- Sample blank roof file structure
- CLI tool that creates a new blank roof inspection folder
- Basic completeness scoring for inspection JSON files

## Folder Structure Generated by the CLI

```text
my-roof-inspection/
```

## Why This Exists

Residential roof documentation is often spread across photos, estimates, notes, claim documents, reports, and unstructured PDFs.

RoofFile Protocol provides a simple shared structure for organizing roof inspection files so they are easier to generate, review, exchange, and understand by contractors, property owners, adjusters, attorneys, software developers, and AI systems.

## Use Cases

RoofFile Protocol can be used for:

- Residential roof inspection documentation
- Hail damage documentation
- Wind damage and wind uplift documentation
- Claim-ready roof file organization
- Photo manifest tracking
- Structured roof report templates
- AI-readable inspection data
- Documentation handoff between contractors, homeowners, and claim participants
