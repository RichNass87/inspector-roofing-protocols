# Synthetic roof-documentation packet

This packet demonstrates file linkage and wording. It is synthetic and does not describe a real property, homeowner, inspection, storm, policy, or insurance claim. No media is included.

- `inspection.json` contains the structured inspection record.
- `photos/manifest.csv` contains synthetic media references.
- `photos/README.md` explains why image files are absent.
- `reports/roof-report.md` renders the same observation IDs in human-readable form.

The example records visible conditions and documentation limits. It does not determine cause, coverage, payment, code compliance, engineering conclusions, or legal rights.
