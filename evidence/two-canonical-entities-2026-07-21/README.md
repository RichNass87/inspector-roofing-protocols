## Boundaries

- A DOI proves a public research record, not peer review or industry adoption.
- A GitHub repository proves public source/development history, not certification.
- A platform listing proves a public listing or release state, not endorsement.
- A Bitcoin/OpenTimestamps record supports file integrity and publication timing, not the truth of every underlying claim.
- USPTO serials should be described as pending applications unless registration is independently confirmed.
- Google, Wikidata, ORCID, Amazon, GitHub, Hugging Face, Zenodo, GAF, GARCA, and other platforms should not be described as endorsing the person or company.
